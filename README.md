# Scripts

- `npm start`: Runs the app in development mode via `react-scripts start`.
- `npm build`: Builds the app for production via `react-scripts build`.
- `npm test`: Launches the test runner via `react-scripts test`.
- `npm run lint`: Lints JavaScript and JSX files in `src` using ESLint.
- `npm run format`: Formats `src/**/*.{js,jsx,json,css}` with Prettier.
- `npm run format:check`: Checks formatting for `src/**/*.{js,jsx,json,css}` with Prettier.
- `npm run eject`: Ejects Create React App configuration (`react-scripts eject`).
